# Contributing to DevForge AI

Thank you for your interest in contributing to the most advanced autonomous software factory. We operate as an open engineering organization.

## 🤝 How to Contribute

### 1. Agent Development
We are always looking to improve our multi-agent intelligence. If you have ideas for new specialized agents (e.g., QA Agent, UX Agent), please open an issue or PR.

### 2. Core Governance
Help us refine the `GovernancePolicy` and `EvolutionEngine`. We aim for the highest standards of architectural integrity.

### 3. Dashboard UI
Our Mission Control dashboard is built with React and Vanilla CSS. We welcome UI/UX improvements that enhance real-time observability.

## 🛠️ Local Development

1.  **Clone**: `git clone https://github.com/Raphasha27/devforge-ai`
2.  **Environment**: Create a `.env` file with `OPENAI_API_KEY` and `GITHUB_TOKEN`.
3.  **Run**: `docker-compose up --build`

## 📜 Code of Conduct
We follow a strict professional standard. Autonomous intelligence requires high-quality, predictable human oversight.
